# TCP Chat Server and Client.

Run the server: 
`sbt server`

Run the client connecting to the server:

`sbt scott`

`sbt jack`

`sbt adam`
   

Supported commands:

`text` broadcast to all connected users.

`/users` list active users.

`/dm:adam:text` sent dm to an active user.  

`/quit` disconnect and terminate client.
